Это еще одна версия миссии ”Power Supply” созданной по задумке oduvan-а
Вам предстоит расположить электростанции так, чтобы обеспечить электричеством все города.

Структура линий электропередач и радиус действия выданных электростанций - входные данные.
Одна электростанция снабжает электричеством город, в котором она размещена и соседние города, в соответствии с ее радиусом действия.
Вы должны вернуть словарь, ключи которого представляют города, где вы установили электростанции,
а значения - радиусы действия использованных электростанций.

ЗАМЕЧАНИЕ:

В вашем распоряжении всегда есть достаточное количество электростанций. Поэтому не всегда обязательно использовать их все.

Пример: 

power_plants({('A', 'B'), ('B', 'C'), ('C', 'D'), ('D', 'E')}, [2]) == {'C': 2}
power_plants({('A', 'B'), ('B', 'C'), ('C', 'D'), ('D', 'E'), ('E', 'F')}, [1, 1]) == {'B': 1, 'E': 1}
power_plants({('A', 'B'), ('B', 'C'), ('A', 'D'), ('B', 'E')}, [1, 0]) == {'B': 1, 'D': 0}

Входные данные:

  • Междугородняя сеть линий электропередач, представленная множеством связей, где каждая связь - кортеж из двух соединенных городов.
  • Радиусы действия электростанций в виде списка целых чисел.

Выходные данные:

  • Словарь размещений, в котором ключ - это город с размещенной электростанцией, а значение - радиус действия использованной станции.

Предварительные условия:

  • 3 ≤ количество городов ≤ 50
  • 1 ≤ количество электростанций
  • радиус действия электростанции ≥ 0
